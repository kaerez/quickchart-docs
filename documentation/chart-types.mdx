# Chart Types

import Admonition from '@theme/Admonition';
import ChartExample from './components/ChartExample';
import Image from './components/Image';

For a variety of useful editable examples, see the **[chart gallery](https://quickchart.io/gallery/)**.

Changing chart types can be very simple. By changing `type: bar` to `type: line`, for example, we can instantly produce an equivalent line graph:

## Bar chart

See [Bar Chart documentation](https://www.chartjs.org/docs/2.9.4/charts/bar.html)

<ChartExample
  noLazyLoad
  alt="Example bar chart"
  config={`{
  type: 'bar',
  data: {
    labels: ['January', 'February', 'March', 'April', 'May'],
    datasets: [
      { label: 'Dogs', data: [50, 60, 70, 180, 190] },
      { label: 'Cats', data: [100, 200, 300, 400, 500] },
    ],
  },
}`}
/>

<Admonition type="info">Set `type` to `horizontalBar` for a horizontal bar chart.</Admonition>

## Line graph

See [Line chart documentation](https://www.chartjs.org/docs/2.9.4/charts/line.html)

<ChartExample
  alt="Example line chart"
  showSquashedUrl

  data: {
    labels: ['January', 'February', 'March', 'April', 'May'],
    datasets: [
      {

        label: 'Cats',
        data: [100, 200, 300, 400, 500],
        fill: false,
        borderColor: 'green',
      },
    ],
  },
}`}
/>

## Pie chart

See [Pie chart documentation](https://www.chartjs.org/docs/2.9.4/charts/doughnut.html). This chart also uses the [datalabels plugin](https://chartjs-plugin-datalabels.netlify.app/).

Learn how to [customize pie chart labels](/documentation/chart-js/custom-pie-doughnut-chart-labels/).

<ChartExample
  showSquashedUrl
  config={`{
  type: 'pie',
  data: {
    labels: ['January', 'February', 'March', 'April', 'May'],
    datasets: [{ data: [50, 60, 70, 180, 190] }],
  },
}`}
/>

## Doughnut chart

See [Doughnut Chart documentation](https://www.chartjs.org/docs/2.9.4/charts/doughnut.html). You can also use options from the Chart.js [doughnutlabel plugin](https://github.com/ciprianciurea/chartjs-plugin-doughnutlabel), in addition to the [datalabels plugin](https://chartjs-plugin-datalabels.netlify.app/).

Learn how to [customize doughnut chart labels](/documentation/chart-js/custom-pie-doughnut-chart-labels/).

<ChartExample
  alt="Example doughnut chart"
  showSquashedUrl
  config={`{
  type: 'doughnut',
  data: {
    labels: ['January', 'February', 'March', 'April', 'May'],
    datasets: [{ data: [50, 60, 70, 180, 190] }],
  },
  options: {
    plugins: {
      doughnutlabel: {
        labels: [{ text: '550', font: { size: 20 } }, { text: 'total' }],
      },
    },
  },
}`}
/>

The doughnut chart type is very flexible.  For example, you can use a half-doughnut to make a gauge-style chart.

<ChartExample
  alt="Example gauge chart"
  showSquashedUrl
  config={`{
  type: 'doughnut',
  data: {
    datasets: [
      {
        data: [24, 66],
        backgroundColor: ['green', '#eee'],
        label: 'Dataset 1',
        borderWidth: 0,
      },
    ],
    labels: ['A', 'C'],
  },
  options: {
    circumference: Math.PI,
    rotation: Math.PI,
    cutoutPercentage: 75,
    layout: {
      padding: 40,
    },
    legend: {
      display: false,
    },
    plugins: {
      datalabels: {
        color: '#404040',
        anchor: 'end',
        align: 'end',
        formatter: (val) => val + '%',
        font: {
          size: 25,
          weight: 'bold',
        },
      },
      doughnutlabel: {
        labels: [
          {
            text: '\\nYour status is',
            font: {
              size: 20,
            },
          },
          {
            text: '\\nhealthy',
            color: '#000',
            font: {
              size: 25,
              weight: 'bold',
            },
          },
        ],
      },
    },
  },
}`}
/>

## Polar chart

Polar charts are like pie and doughnut charts, except each segment has the same angle and the radius of the segments varies based on the value. See [Polar Area Chart documentation](https://www.chartjs.org/docs/2.9.4/charts/polar.html).

<ChartExample
  alt="Example polar chart"
  showSquashedUrl
  config={`{
  type: 'polarArea',
  data: {
    labels: ['January', 'February', 'March', 'April', 'May'],
    datasets: [{ data: [50, 60, 70, 180, 190] }],
  },
}`}
/>

## Bubble chart

Bubble is similar to Scatter, except the `r` variable defines bubble radius in pixels. [Bubble Chart documentation](https://www.chartjs.org/docs/2.9.4/charts/bubble.html)

<ChartExample
  alt="Example bubble chart"
  showSquashedUrl
  config={`{
  type: 'bubble',
  data: {
    datasets: [
      {
        label: 'Data 1',
        data: [
          { x: 1, y: 4, r: 9 },
          { x: 2, y: 4, r: 6 },
          { x: 3, y: 8, r: 30 },
          { x: 0, y: 10, r: 1 },
          { x: 10, y: 5, r: 5 },
        ],
      },
    ],
  },
}`}
/>

## Speedometer Charts

See customization options at [chartjs-gauge](https://github.com/haiiaaa/chartjs-gauge).

<ChartExample
  alt="Example speedometer chart"
  showSquashedUrl
  config={`{
  type: 'gauge',
  data: {
    datasets: [
      {
        value: 50,
        data: [20, 40, 60],
        backgroundColor: ['green', 'orange', 'red'],
        borderWidth: 2,
      },
    ],
  },
  options: {
    valueLabel: {
      fontSize: 22,
      backgroundColor: 'transparent',
      color: '#000',
      formatter: function (value, context) {
        return value + ' mph';
      },
      bottomMarginPercentage: 10,
    },
  },
}`}
/>

## Box Plots and Violin Charts

You can use `violin`, `boxplot`, `horizontalBoxPlot`, and `horizontalViolin` chart types. View the options at [chartjs-chart-box-and-violin-plot](https://github.com/datavisyn/chartjs-chart-box-and-violin-plot) for customization details.

<ChartExample
  alt="Example violin chart"
  showSquashedUrl
  config={`{
  type: 'violin',
  data: {
    labels: [2012, 2013, 2014, 2015],
    datasets: [
      {
        label: 'Data',
        data: [
          [12, 6, 3, 4],
          [1, 8, 8, 15],
          [1, 1, 1, 2, 3, 5, 9, 8],
          [19, -3, 18, 8, 5, 9, 9],
        ],
        backgroundColor: 'rgba(56,123,45,0.2)',
        borderColor: 'rgba(56,123,45,1.9)',
      },
    ],
  },
}`}
/>

<Admonition type="tip">
  For best results, add a scatter plot to your box or violin chart.
</Admonition>

## Financial Charts

QuickChart supports Candlestick and OHLC charts using the [chartjs-chart-financial](https://github.com/chartjs/chartjs-chart-financial) plugin. These are commonly used for technical analysis and stock charts.

Note that you'll have to select Chart.js version 3+ to use these plugins. [View example configuration](https://quickchart-urls.ty.workers.dev/ohlc-stock-chart)

## Sparklines

A sparkline is a special case of line graph with axes and other labeling removed. All [line graph options](https://www.chartjs.org/docs/2.9.4/charts/line.html) can be applied. Read more about the [sparkline API](/documentation/sparkline-api/).

<ChartExample
  alt="Example sparkline"
  showSquashedUrl
  height={100}
  width={300}
  config={`{
  type: 'sparkline',
  data: { datasets: [{ data: [140, 60, 274, 370, 199] }] },
}`}
/>

## Sankeys

A sankey diagram is a flow diagram that illustrates the proportions between various stages of a flow. To learn more, see [chartjs-chart-sankey documentation](https://github.com/kurkle/chartjs-chart-sankey).

<ChartExample
  version="3"
  alt="Example sankey"
  showSquashedUrl
  config={`{
  type: 'sankey',
  data: {
    datasets: [
      {
        data: [
          { from: 'Step A', to: 'Step B', flow: 10 },
          { from: 'Step A', to: 'Step C', flow: 5 },
          { from: 'Step B', to: 'Step C', flow: 10 },
          { from: 'Step D', to: 'Step C', flow: 7 },
        ]
      },
    ],
  },
}`}
/>

A more elaborate sankey example with custom colors and position is available [here](https://quickchart-urls.ty.workers.dev/sankey-diagram).

## Progress bars

A progress bar is a special case of a horizontal bar chart with axes and other labeling removed. All [bar options](https://www.chartjs.org/docs/2.9.4/charts/bar.html) can be applied.

The first dataset specifies the number shown. By default, this number is a percentage and the total value is 100. If your data is not out of 100, add a second dataset to override the total.

<ChartExample
  alt="Example progress bar"
  showSquashedUrl
  width={300}
  height={50}
  config={`{
  type: 'progressBar',
  data: {
    datasets: [
      {
        data: [50],
      },
    ],
  },
}`}
/>

## Mixed charts

Combine charts together by specifying multiple `type`s:

<ChartExample
  alt="Example mixed chart (line and bar)"
  showSquashedUrl
  config={`{
  type: 'bar',
  data: {
    labels: ['January', 'February', 'March', 'April', 'May'],
    datasets: [
      { label: 'Dogs', data: [50, 60, 70, 180, 190] },
      { label: 'Cats', data: [100, 200, 300, 400, 500] },
      {
        type: 'line',
        fill: false,
        label: 'Potatoes',
        data: [100, 400, 200, 400, 700],
      },
    ],
  },
}`}
/>
